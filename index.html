Projects — Internship & Academic

Trailer Add

Trailer Add is a mobile application built on Titanium SDK which is a cross platform tool. This application is used by more than 20 drivers per day to get the trailer moved from one location to another. To keep track of all these they are using this application by signing in and entering some key information and submitting. After submitting, all the data is wrapped up and sent to database. This application also has offline submitting. The application has been deployed on iOS devices.

Trailer View

Trailer View is a mobile application built on Titanium SDK. This application is a continuation to Trailer Add application. The application has more than 10,000 trailer VIN numbers to search for which was used to track the trailer using details such as - Driver Name, Lot No., Date, Latitude and Longitiude. The application maps each trailer on a map view for comfortable viewing. The application also has offline viewing. The application has been deployed on iOS devices.
